## How To Compile

0. Install Node

Install it in the original site, or if you're in Windows, you can use the ocmmand line, and type,

```
winget install OpenJS.NodeJS
```

1. Install all dependencies

```
npm install
```

2. Compile to a single .HTMl file

```
npm run build
```
