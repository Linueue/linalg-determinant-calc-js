# Determinant Calculator

## Using Cofactor Expansion

## How To Compile

1. Install Node (Skip if you already have Node)

Install it in the [original site](https://nodejs.org/en/download), or if you're in Windows, you can use the command line, and type,

```
winget install OpenJS.NodeJS
```

2. Clone the repo

```
git clone https://github.com/Linueue/linalg-determinant-calc-js
```

3. Go to the repo's directory

```
cd linalg-determinant-calc-js
```

4. Install all dependencies

```
npm install
```

5. Compile to a single .HTML file

```
npm run build
```
